# Furniture-sales
Client-server application that implements RESTful API architecture and interaction with PostgreSQL database via web interface.

# Dependencies
- python 3.10
- "connexion[swagger-ui,flask]"
- flask-marshmallow[sqlalchemy]==0.14.0
- psycopg2-binary
- httptools
- python-dotenv
- uvicorn
- watchfiles
- websockets
